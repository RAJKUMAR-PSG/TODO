# Todo Application (Spring + JDBC)

This is a simple Todo application built using **Spring Boot and JDBC**, without using JPA or Hibernate.

## Features
- Create Todo
- Get all Todos
- Get Todo by ID
- Update Todo
- Delete Todo

## Tech Stack
- Java 17
- Spring Boot
- Spring JDBC
- MySQL
- Maven

## Purpose
This project is built to understand how database operations work **without ORM**, before moving to JPA/Hibernate.
